Setup:

Add Google Map API-key in
1. mapapp/map/views.py  (line 32)
2. mapapp/map/templates/map/path2.html  (line 2)

Add Library
pip install googlemaps


Running:

python manage.py runserver

Goto:
http://127.0.0.1:8000/map/

Fill the Longitutes and Latitutes

Examples :

28.536757,28.740200,28.698777
77.082208,77.163607,77.305298

28.750739,28.696820,28.725388,28.733435
77.207597,77.082076,77.123743,77.315293
